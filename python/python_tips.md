## `python -m pip`を使う理由
----------------------------------------
- `python`を付けることで、使用するPythonインタプリタを指定することができる。(`-m`は実行するモジュールの指定)
- Windowsの場合、`python -m pip install --upgrade pip`でpip自体を更新できる。(pip単独で実行した場合、pipが使用状態となり更新できない)
